# 🍻这里是功能介绍~
因为是一个很基础的日程app，所以当前的功能还不算太多。所有功能大致包括：

**日程**：定义所有已知时间的事件为“日程”。每个日程可设置tag【如工作、学习等】；所有日程均包括【单次日程，可设置起止时间】【同一天多次日程，可添加多个时间段】【跨天重复日程，可设定起止日期和每次发生的时间段】。如果是瞬时事件，米斯特温德支持只设置发生时间。日程具有提醒功能，届时会以banner的形式在通知栏进行通知，理想情况下会被手表/手环等便携设备抓取到并震动提示

**待办todo**：定义所有未来需要完成，但是未知时间的事件为“todo”。

**时间轴**:所有新建的日程与todo会同步到时间轴中，本意是时间轴左侧是todo/日程，右侧是done list，但是现在呈现的结果好像不太对，先将就着用，嘿嘿（。）

**番茄钟倒计时**：即专注计时，目前支持关联todo。所有的专注倒计时达到一定时长后，结束时会同步在时间轴的donelist中

**重要日**：即“倒计时功能”，比如【距离生日还有100天】

**日历可视化**：针对不同场景与需求，米斯特温德支持三种日程可视化，每天的日程与todo均会出现在日历中，其中已知事件的日程在上，未知时间的todo会以todo box的形式存放在最下面

**习惯打卡**：可以设置频率（不过是自然语言输入，目前没有提醒功能）。针对类似体重、身高、阅读、追番等打卡的场景，可以设置【数值】类型与单位；针对心情打卡场景，支持设置心情指数；针对吃饭、旅游、电影、游戏等打卡场景，支持选择图片属性，后续会优化这种属性的照片墙功能

**照片墙**：针对图片类型的打卡生成的一种统计视图。现在应该只有一个简单的实现，大概率会有bug

**小工具**：任意界面上滑/点击屏幕下方小字可进入小工具次元。目前支持网址密码收藏备忘、灵感随手记功能【这个功能好像有下滑的bug，后续优化】

**小组件**：简单的占位实现。可以在桌面设置小组件，目前支持两种日历视图。【不过界面长的很丑米斯特温德本人也不想用】

**tag**：所有元素【包括日程、todo、番茄钟倒计时、重要日、打卡】均支持共享tag，tag可以在首页自定义，每一个新建tag会随机一种颜色
