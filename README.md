# 米斯特温德

米斯特温德是一个本地优先的复古简洁日程管理 Android 应用，面向华为 Mate70 / 鸿蒙 4.2 的 Android 应用兼容环境设计。

## 当前功能

- 定时日程：支持单个时间点、同一天多个时间段、Routine 每天重复并选择结束日期。
- Todo Box：未确定具体时间的待办集中显示，可完成、上移、下移。
- 独立时间轴：底部导航提供固定 24 小时时间轴，全天 Todo、定时日程与按完成时间归档的 Done 会同步呈现。
- 专注计时：支持 1–240 分钟自定义时长、Tag 选择、关联 Todo、暂停/继续/重置；结束后自动归档关联 Todo（自由专注会生成完成记录），后台由本地闹钟保证落库。
- 日程可视化：月/周/三日视图会展示每天涉及的 Tag 色条；月视图支持前后翻月，点选日期后联动显示当天全部日程。
- 日程操作：完成时有划线动画，完成后变灰并移动到底部；也支持延后 30 分钟、移至明天、手动改期、删除。
- 本地提醒：重要日程可选择提前 10 分钟、30 分钟、1 小时、1 天通知栏提醒。
- 重要日倒计时：显示距离目标日期还有多少天。
- 打卡：可自定义打卡类型，打卡时保存文字和可选图片。
- 网址收藏夹：保存网址、账号、密码和备注，密码使用 Android Keystore 加密后再写入 SQLite。
- 随手记：通过对话式窗口发送文字或图片，文字存入 SQLite，图片复制到固定的 `./mstwd/quick-notes/images` 目录。
- 桌面小组件：月视图为月历，周视图为周历，下方都有 Todo Box；日程可在小组件中完成、延后或点按打开管理选项。
- App 内导航：底部为 `首页 / 时间轴 / 日历 / 打卡`，小工具从底部小把手上拉进入；页面、卡片、弹窗及状态切换均带有轻量动效。
- 本地存储：业务数据统一保存到应用沙盒内的固定 `./mstwd` 文件夹；数据库为 `mstwd.db`，打卡图片位于 `checkins/images`，随手记图片位于 `quick-notes/images`。长按右上角头像可查看并复制实际路径。

## 构建方式

1. 用 Android Studio 打开本目录。
2. 等待 Android Gradle Plugin 和 Android SDK 同步完成。
3. 连接手机，选择 `app` 运行。
4. 首次打开时允许通知权限，提醒功能才能在通知栏出现。

当前工程不依赖 AndroidX 或第三方库，主要由原生 Java、XML、SQLite、AlarmManager、Notification、AppWidget 组成。
